* [MySQL]()

### Installation


1. Clone this repo 
    ```sh
    git clone git@github.com:Montyking20/E-Commerce-BackEnd.git
    ```
2. Install NPM packages
    ```sh
     npm install
    ```
3. Source the database. 
    ```sh
    source db/schema.sql
    ```
4. Seed the DB
    ```sh
    npm run seed
    ```

## Usage

Once installed run `npm run start`
